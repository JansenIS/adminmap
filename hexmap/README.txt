Запуск:
1) Можно открыть index.html напрямую (file://).
2) Если браузер ругается, запусти локальный сервер:
   - Python:  python -m http.server 8000
   - Node:    npx http-server -p 8000
и открой http://localhost:8000/

Управление:
- Колесо мыши: зум
- Перетаскивание: пан
- Двойной клик: сброс масштаба
- Режим наведения: Гекс / Провинция
